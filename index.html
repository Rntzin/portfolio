GymPass App

Este aplicativo está em andamento seguindo os princípios SOLID, até o momento ele permite aos usuários se cadastrarem, autenticarem e gerenciarem suas atividades em academias, com funcionalidades inspiradas no GymPass. Usuários podem consultar seu perfil, histórico de check-ins e academias próximas, além de realizar check-ins. Administradores podem cadastrar academias e validar check-ins de usuários.

Tela de Login

Esta aplicação oferece uma tela de login simples e funcional, onde os usuários podem se autenticar utilizando suas credenciais. O design é responsivo e adaptável para diferentes dispositivos, foi utilizado React com Typescript para construção da interface, Chakra UI para estilização e design responsivo, React Hook Form para gerenciamento de formulários, Zod para validação de campos do formulário, JWT (JSON Web Token) para autenticação segura.
